# H2O.ai Docker Wrapper

A simple wrapper for Open Source version of [h2o.ai](https://www.h2o.ai).

## Installation

Application is published as a docker image.

```bash
docker pull huanga/h2o-ai
```


## Usage

```bash
docker run \
    -p 54321:54321 \
    -p 54322:54322 \
    huanga/h2o-ai
```
